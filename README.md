# Personal Portfolio

This is my personal portfolio website — a clean, responsive site built to showcase my skills, projects and contact info. It includes a typewriter effect for dynamic text display (using an npm package) to make the intro section more engaging.

## Live Demo
https://aliraza-personalportfolio.netlify.app/

**Features:**
- Responsive layout (works well on desktop and mobile)  
- Intro section with typewriter effect for name/title  
- Sections for About, Skills, Projects, Contact (or similar)  
- Clean HTML, CSS, JS structure — easy to customize and extend  
- Lightweight and minimal dependencies  

## How to Use
1. Clone or download this repository:  
   ```bash
   git clone https://github.com/Muhammad-Ali-16/personal-portfolio.git
   ```  
2. Install dependencies (Node.js packages):  
   ```bash
   npm install
   ```  

3. Run the application:  
   ```bash
   npm run dev
   ```  
4. Open the app in your browser or terminal (depending on your setup)
5. 
**Project Files (typical):**
- `index.html` — main page / UI  
- `style.css` — styling for the layout and responsiveness  
- JS files — includes logic for the typewriter effect and any interactivity  
- Asset folders (images, icons, etc.) — for profile picture, project thumbnails, etc.
